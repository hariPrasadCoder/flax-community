# Flax Community

Empowering Engineers in the AI Era.

## About

Flax Community is a public collection of real stories from senior engineers in Data/AI on how they work, lead, and ship as AI becomes part of everyday engineering.

## Features

- **Homepage**: Introduction to the community and its mission
- **AI Impact Contributors**: A campaign featuring 25 senior engineers sharing their stories

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS (via CDN)
- React Router
- Lucide React (icons)
- Cal.com Embed (for scheduling)
